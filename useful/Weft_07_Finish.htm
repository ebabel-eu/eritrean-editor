WEFT 3 release notes


Finished!

When you see this screen the embedding process is complete, and you should move on to testing the embedded fonts created by WEFT and the Web pages it has modified.

You may also wish to save the current WEFT settings by selecting 'Save' from WEFT's 'File' menu or by clicking the 'Save' button.


Options on this screen

Back : Use this button to return to the 'Publish Web pages' screen.

Finish : Click here to exit the WEFT wizard and return you to the main WEFT screen.

Cancel : This button has the same effect as 'Finish'. Clicking this button will exit the WEFT wizard and return you to the main WEFT screen.
